## Предусловия:

-  пользователь находится в разделе "Все проекты"

-  в системе присутствуют проекты с различными статусами, наставниками, тегами и уровнями сложности.

## Шаги:

1. Посетитель открывает главную страницу.

2. Система отображает страницу со списком из 10 случайно отобранных карточек проектов в состоянии «В архиве»

3. Пользователь использует пагинацию для листания страницы списка проектов.

4. Пользователь нажимает на кнопку «Результаты проекта»

5. Осуществляется переход на страницу проекта и открывается вкладка «Результаты проекта»

## Ожидаемый результат:

Система отображает только проекты, в состоянии «В архиве».

Каждый раз при обновлении страницы список отображаемых проектов обновляется случайным образом.

Пользователь может ознакомиться с результатами проекта.
