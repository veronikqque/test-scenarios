title: Создание черновика новой проектной заявки
---

## Предусловия:

- в данный момент идет набор проектных заявок (в соответствии с бизнес правилом)

-  пользователь авторизован как “сотрудник”

-  пользователь находится в разделе создания проектных заявок

## Шаги:

1. Пользователь находит и нажимает на кнопку "Создать проект"

2. В открывшемся окне пользователь выбирает тип проекта:
   - "Новый проект"
   - "Продолжить старый" (при выборе этого варианта отображается дополнительный выбор проекта для продолжения)

3. Пользователь заполняет:
   - Данные о проекте (название/описание) — максимум 255 символов
   - Цель проекта — максимум 255 символов
   - Заказчик (например, "ЦЭО, Лукьянов Н.Д.")
   - Источник темы (выбор из предложенных вариантов)

4. Пользователь выбирает:
   - Длительность проекта:
     - 1 семестр (осень)
     - 1 семестр (весна)
     - 2 семестра (осень и весна)
   - Сложность проекта:
     - Легкий
     - Средний
     - Сложный

5. Пользователь устанавливает Роли в проекте:
   - Видит свое ФИО в качестве "Разработчика проекта" (Разработчиком проекта становится сотрудник ИРНИТУ, создавший проектную заявку через Ярмарку проектов)
   - Видит "Подразделение, к которому будет привязан проект" (Заполняется автоматически в зависимости от выбранного на проекте наставника)
   - Указывает "ФИО преподавателя" на роль "Наставника"
   - При необходимости нажимает кнопку "+ Добавить сонаставника"

6. Пользователь заполняет:
   - Ожидаемый результат — максимум 1000 символов
   - Формируемые навыки — максимум 1000 символов
   - Описание проекта — максимум 3000 символов

7. Пользователь видит "Направления (специальности), участников проекта"

8. Пользователь заполняет пункт 6 "Приглашённые направления (специальности), участников проекта" (Можно пригласить любые специальности, студенты этих направлений самостоятельно смогут подать заявку на участие в Вашем проекте):
   - Нажимает "Редактировать специальности +"
   - Выбирает нужные направления

9. Пользователь выбирает "Навыки, которые необходимы на проекте":
   - Нажимает "Редактировать навыки +"
   - Выбирает необходимые навыки

10. Далее пользователь нажимает на кнопку "Сохранить черновик"


## Ожидаемый результат:

Заявка отображается в списке "Мои заявки" с статусом "Черновик"
